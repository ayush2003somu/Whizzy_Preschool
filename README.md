# 🌟 Whizzy Preschool Website

A modern, responsive, and user-friendly preschool website developed for a client.  
This project focuses on creating a visually appealing and informative platform for parents to explore the school’s environment, facilities, and programs.

---

## 🚀 Live Demo
🔗 https://whizzy.co.in  

---

## 📌 Features

- 🎯 Fully responsive design (Mobile + Tablet + Desktop)
- 🎨 Clean and modern UI/UX
- 🏫 Sections for About, Courses, Gallery, and Contact
- 📸 Image gallery showcasing school activities
- 🧒 Child-friendly design and layout
- 🔐 Secure HTTPS deployment
- 📍 Integrated Google Maps location
- 📱 Mobile navigation (hamburger menu)
- ⚡ Smooth animations and transitions
- 🌐 Custom domain integration

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid + Media Queries)  
- JavaScript (Vanilla JS)  
- Firebase Hosting
